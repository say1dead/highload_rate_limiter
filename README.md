Распределённая система ограничения запросов (Rate Limiting)
– Разработал микросервисную систему на ASP.NET Core (gRPC) для управления
пользователями и лимитами запросов.
– Реализовал сервисы чтения и записи лимитов с использованием MongoDB,
кэшированием в памяти и обновлением через Change Streams. Работа с
пользователями осуществлялась с помощью PostgreSql.
– Настроил обработку событий через Apache Kafka и контроль превышения лимитов
с записью в Redis (TTL).
– Добавил gRPC-интерцепторы для валидации запросов и блокировки
пользователей при превышении лимита.
– Развернул систему в Docker Compose; применил асинхронность, DI и
конфигурацию через IOptions.
